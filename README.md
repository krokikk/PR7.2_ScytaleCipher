### Применяемые виды тестирования:

В процессе разработки и проверки были использованы следующие виды тестирования:
- Модульное тестирование (Unit Testing) — проверка методов шифрования и дешифрования с использованием NUnit;
- Интеграционное тестирование — проверка совместной работы модулей шифрования и пользовательского интерфейса;
- Ручное тестирование — проверка нефункциональных требований (интерфейс, удобство, обработка ошибок);
- Регрессионное тестирование — повторная проверка после исправления выявленных ошибок.
